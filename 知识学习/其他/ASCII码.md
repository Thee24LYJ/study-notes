1.ASCII 码表中数字 0~9 升序排列，字母 A~Z、a~z 是按顺序依次排列的，相邻字符的码值依次递增，大写与小写字母的 ASCII 码相差 32

2.特殊记忆

| 字符  | 对应 ASCII 码 | 十六进制   |
| --- | ---------- | ------ |
| 0~9 | 48~57      | 30~39H |
| A~Z | 65~90      | 41~5AH |
| a~z | 97~122     | 61~7AH |
| 空格  | 32         | 20H    |

其实只要记住数字或字母的首位对应的 ASCII 码即可,其他的均可以以此推算
